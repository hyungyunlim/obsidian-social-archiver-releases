# Contributing

This repository is a **public source mirror** of the internal development repo,
synced automatically on each release.

## How to contribute

- **Bug reports / Feature requests**: Please open an Issue — all feedback is welcome!
- **Security issues**: Open an Issue or email directly. Security fixes will be prioritized.
- **Code PRs**: Source changes are managed internally. PRs with code changes
  will generally be closed, but valuable insights will be credited and applied
  through the internal workflow.

Thank you for your interest in Social Archiver!
